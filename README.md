# Visualize it

Tell us your story, and we'll bring it to life visually with AI.

[Deployed at Netlify](https://visualize-it-with-ai.netlify.app/)

## Tech stack

Built with:

- [Vite](https://vitejs.dev/) - Next generation frontend tooling
- [React](https://react.dev/) - JavaScript library for web interfaces
- [TypeScript](https://www.typescriptlang.org/) - JavaScript with syntax for types
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- [React Icons](https://react-icons.github.io/react-icons/) - Popular icons in React projects
